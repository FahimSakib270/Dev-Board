Question-3: What is event bubbling in the DOM?

Event Bubbling is a technique in JavaScript where an event is first triggered on the target element and then propagates up through its parent elements in the DOM. Instead of handling events separately for each element, the event automatically moves from the child element to its parents, allowing a single event listener to handle multiple elements efficiently.

Question-4: What is event delegation in the DOM?

Event Delegation is a technique in JavaScript where an event is handled by a parent element instead of attaching event listeners to multiple child elements. This allows the parent to detect events from its child elements, even if they are added dynamically. Instead of adding separate event listeners to each child element, a single event listener is attached to the parent.

Question-5: How do you create, add, and remove elements using JavaScript?

Here are the way to create , add and remove elements in JavaScript-

  1. Create : Use document.createElement() to create a new element.
  2. Add : Use appendChild(), append(), prepend(), or insertBefore() to place the element in the DOM.
  3. Remove : Use removeChild() or remove() to delete an element from the DOM.
